# Sky Rogue 게임 플로우

문서 버전: 0.1  
최종 수정: 2026-07-03  
상태: 타이틀 → 로비 → 인게임 프로토타입 반영

## 전체 흐름

실행 → 타이틀 화면 → 로비(챕터 리스트) → 챕터 입장 → 웨이브 진행 → 결과 화면 → 로비

## 1. 실행

- 게임 리소스와 저장 데이터를 불러온다.
- 최초 실행 시 기본 설정과 신규 플레이어 데이터를 생성한다.
- 로딩 완료 후 타이틀 화면으로 이동한다.

## 2. 타이틀 화면

목적: 게임 진입과 계정·데이터 준비 상태 확인.

- 게임 로고와 대표 드래곤을 표시한다.
- `화면을 터치하여 시작` 또는 `게임 시작` 버튼을 제공한다.
- 설정, 공지, 데이터 복구 기능은 추후 추가한다.
- 입력 시 로비로 이동한다.

## 3. 로비 — 챕터 리스트

목적: 플레이할 콘텐츠와 출전 드래곤을 선택하는 메인 허브.

- 현재 열린 챕터 목록
- 챕터별 진행도와 최고 기록
- 권장 전투력 또는 난이도
- 선택한 플레이어 드래곤
- 출전 버튼
- 향후 상점, 드래곤 관리, 성장 메뉴 진입점

챕터를 선택하고 `출전`을 누르면 챕터 입장 화면으로 이동한다.

## 4. 챕터 입장

목적: 전투 전 목표와 구성을 확인하고 게임 리소스를 준비한다.

- 챕터 이름과 배경
- 총 웨이브 수
- 등장 적과 보스 예고
- 추천 속성 또는 주요 위험 패턴
- 출전 드래곤과 장착 요소 확인
- 짧은 로딩 후 전투 시작

## 5. 웨이브 진행

목적: 챕터의 핵심 플레이 구간.

1. 웨이브 시작 안내
2. 일반 적 편대 전투
3. 경험치 획득과 레벨업 강화 선택
4. 웨이브 상승
5. 엘리트 또는 보스 전투
6. 최종 보스 처치 시 챕터 클리어

플레이어 HP가 0이 되면 즉시 실패 결과로 이동한다.

## 6. 결과 화면

### 클리어

- 챕터 클리어 표시
- 생존 시간, 점수, 처치 수
- 선택했던 강화와 최종 빌드
- 획득 보상
- 신규 최고 기록 및 다음 챕터 해금

### 실패

- 도달 웨이브와 보스 진행도
- 생존 시간, 점수, 처치 수
- 선택했던 강화와 최종 빌드
- 획득 가능한 일부 보상

공통 버튼:

- `로비로`
- `재도전`

`로비로` 선택 시 진행 결과를 저장하고 로비로 돌아간다. `재도전`은 같은 챕터 입장 단계로 이동한다.

## 화면 상태 전환

| 현재 상태 | 입력·조건 | 다음 상태 |
|---|---|---|
| 실행 | 로딩 완료 | 타이틀 |
| 타이틀 | 시작 입력 | 로비 |
| 로비 | 챕터 선택 후 출전 | 챕터 입장 |
| 챕터 입장 | 로딩 완료 | 웨이브 진행 |
| 웨이브 진행 | 최종 보스 처치 | 클리어 결과 |
| 웨이브 진행 | 플레이어 HP 0 | 실패 결과 |
| 결과 | 로비로 | 로비 |
| 결과 | 재도전 | 챕터 입장 |

## 프로토타입 반영 순서

1. [완료] 제공 이미지 기반 타이틀 화면과 START 연결
2. [완료] 제공 이미지 기반 로비 화면과 Play 연결
3. [완료] Play에서 기존 인게임 진입
4. [완료] 결과에서 로비 복귀와 재도전 구현
5. [예정] 로비 챕터 선택 기능 분리
6. [예정] 선택한 챕터 정보를 인게임에 전달
7. [예정] 클리어·실패 결과 화면 세분화
