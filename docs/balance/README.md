# Balance Tables

이 폴더는 게임 밸런스 관리용 테이블을 보관하는 정식 위치입니다.

- `BALANCE_TABLES.md`: 현재 게임 코드 기준으로 사람이 바로 확인하는 밸런스 표
- `build_balance.mjs`: 스프레드시트형 밸런스 문서 생성용 기존 작업 스크립트
- `test_game_logic.mjs`: 밸런스/게임 로직 확인용 테스트 스크립트
- `process_wingmen.py`: 새끼용/보조 드래곤 데이터 처리용 기존 작업 스크립트

임시 작업 폴더였던 `tmp_balance/`는 `.gitignore` 대상이므로, 앞으로 유지해야 하는 밸런스 자료는 이 폴더에 보관합니다.
