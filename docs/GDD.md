# Sky Rogue 게임 기획서

문서 버전: 0.1  
최종 수정: 2026-07-03  
상태: 프로토타입 검증 단계

## 1. 한 줄 소개

드래곤을 조종해 적의 탄막을 피하고, 결정적인 순간에 하강하여 공중 공격을 회피하는 세로형 탑뷰 로그라이트 슈팅 게임.

## 2. 핵심 재미

1. 자동 공격에 집중하지 않고 이동과 회피 판단에 집중한다.
2. 위험한 패턴을 보고 제한된 하강 스킬을 정확한 순간에 사용한다.
3. 매 플레이마다 다른 강화 조합으로 드래곤과 윙맨을 성장시킨다.
4. 다양한 외형과 전투 특성을 가진 드래곤을 수집한다.

## 3. 핵심 플레이 루프

출격 → 적 편대와 전투 → 경험치 결정 획득 → 레벨업 강화 선택 → 웨이브 난이도 상승 → 생존 실패 또는 보스 격파 → 보상 획득 → 드래곤 성장·수집 → 재출격

## 3-1. 전체 화면 플로우

실행 → 타이틀 화면 → 로비(챕터 리스트) → 챕터 입장 → 웨이브 진행 → 결과 화면 → 로비

- 종료는 클리어와 실패 결과를 모두 포함한다.
- 결과 화면에서는 로비 복귀 또는 같은 챕터 재도전을 선택할 수 있다.
- 상세 정의는 `GAME_FLOW.md`를 기준으로 한다.

## 4. 조작과 카메라

- 화면 비율: 9:16 세로형
- 시점: 탑뷰
- 이동: 키보드 또는 터치 드래그
- 공격: 자동
- 액티브 조작: 하강 회피

## 5. 하강 회피

- 사용 즉시 저공 레이어로 이동한다.
- 유지 시간: 3초
- 종료 시 자동 상승한다.
- 재사용 대기시간: 2초
- 하강 중 공중 적, 공중 투사체, 레이저에 피격되지 않는다.
- 하강 중 플레이어와 윙맨은 공격을 중단한다.
- 시각 표현: 캐릭터 축소, 적보다 낮은 렌더링 레이어, 지면 그림자 강화.

이 시스템은 상시 무적기가 아니라 강한 패턴에 맞춰 가끔 사용하는 핵심 판단 스킬이다.

## 6. 전투 패턴

### 플레이어

- 전방 자동 사격
- 연사, 공격력, 다중 발사 강화
- 윙맨 보조 사격

### 적

| 타입 | 외형·구분 | 공격 패턴 | 등장 규칙·상태 |
|---|---|---|---|
| 일반 몬스터 | 기본 파란 와이번 | 플레이어를 조준하는 기본 단발 투사체 | 모든 웨이브의 기본 적 |
| 정예 몬스터 | 일반 와이번 몸체 위에 동작별 은색·금색 갑주 파츠 스프라이트를 합성 | 생성 시 강화 투사체 또는 예고형 직선 레이저 중 하나를 확률 선택 | 1웨이브 0%, 2웨이브 5%, 3웨이브 10%부터 증가하며 최대 28% |
| 중간 보스 | 붉은 갑주 와이번, 대형 HP 게이지 | 현재 전용 조준 레이저 패턴 유지 | 3웨이브마다 등장하며 처치 전까지 화면 상단을 순찰 |
| 보스 | 에메랄드·황금 갑주의 고대 드래곤 | 1단계 화염구·브레스, 2단계 마력 구체·소환, 3단계 화염 폭발·회오리 | 3페이즈 콘셉트 유지, 최종 보스 전투 구현 예정 |

- 조준 단발탄: 현재 플레이어 위치를 향해 한 발 발사
- 강화 투사체: 정예 몬스터가 사용하는 대형 고속 마력탄
- 직선 레이저: 예고선을 표시하고 회전한 방향으로 발사
- 원형 광역탄: 적 중심에서 여러 방향으로 투사체 발사
- 몸체 충돌: 공중 상태에서만 피해

### 정예 파츠 규칙

- 몬스터 타입마다 기본 몸체 스프라이트와 정예 파츠 스프라이트를 각각 등록한다.
- 정예 파츠는 기본 몸체와 동일한 256×256 셀, 동일한 프레임 수와 순서를 사용한다.
- 렌더링 시 몸체와 파츠에 같은 프레임 번호, 위치, 크기, 회전, 피격 플래시를 적용한다.
- 새로운 몬스터를 추가할 때 해당 몬스터 전용 정예 파츠 시트를 함께 제작하며, 공용 아이콘형 갑주를 재사용하지 않는다.

## 7. 성장과 강화

현재 강화:

- 화염 연사
- 용의 심장
- 갈라지는 불꽃
- 바람의 날개
- 고대의 비늘
- 재생의 숨결
- 새끼 드래곤 윙맨

윙맨은 LV.1에서 좌측, LV.2에서 우측에 배치된다. 두 단계 완료 전까지 레벨업 선택지에 항상 노출한다.

## 8. 웨이브

- 기본 웨이브 길이: 25초
- 웨이브 상승 시 적 체력·속도·패턴 밀도가 증가한다.
- 전환 시 중앙 배너로 웨이브를 알린다.
- 장기적으로 일반 웨이브, 엘리트 웨이브, 보스 웨이브로 구분한다.

## 9. 아트 방향

- 밝고 선명한 모바일 카툰 렌더링
- 굵고 정돈된 외곽선과 셀 셰이딩
- 청록색 수면, 밝은 절벽, 풍부한 초목의 판타지 협곡
- 도트 그래픽은 사용하지 않는다.
- 작은 모바일 화면에서도 캐릭터와 투사체 실루엣을 즉시 구분할 수 있어야 한다.

## 10. 캐릭터와 BM 방향

- 플레이어 드래곤은 향후 구매·수집 대상이다.
- 모든 드래곤은 동일한 256×256px, 12프레임 규격을 사용한다.
- 판매 캐릭터 간 판정 크기와 가독성에서 불공정이 생기지 않도록 표준을 유지한다.
- 구체적인 가격, 확률형 상품 여부, 능력치 판매 범위는 아직 확정하지 않는다.

## 11. UI 방향

- 좌측 상단: 초상화, HP, 레벨
- 우측 상단: 점수, 웨이브, 일시정지
- 캐릭터 상단: 전투 중 확인 가능한 HP 바
- 우측 하단: 하강 지속시간과 쿨타임을 표시하는 원형 버튼
- 입체적인 카툰 패널과 높은 명암 대비 사용

## 12. 프로토타입 성공 기준

- 하강 회피를 사용해야 하는 이유가 전투 중 명확하다.
- 세 가지 적 공격 패턴을 시각적으로 즉시 구분할 수 있다.
- 5분 플레이 동안 성장 선택이 최소 3회 이상 발생한다.
- 모바일 화면에서 캐릭터, HP, 위험 예고가 겹치지 않는다.
- 플레이어가 한 판 더 하고 싶은 성장 변화를 체감한다.
