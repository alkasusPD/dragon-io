# 개발 기록

새 작업은 날짜별로 맨 위에 추가한다. 무엇을 바꿨는지뿐 아니라 왜 바꿨는지도 기록한다.

## 2026-07-06

- 손가락이 플레이어 드래곤과 전방 시야를 가리지 않도록 모바일 이동 입력을 화면 좌측 하단 영역으로 분리.
- 좌측 하단을 누른 위치에 플로팅 조이스틱이 나타나는 상대 이동 방식을 유지.
- 조작 복잡도를 낮추기 위해 하강 버튼과 하강 회피 시스템 제거.

## 2026-07-03

- 제공된 9:16 타이틀·로비 이미지를 실제 화면 자산으로 적용.
- START → 로비, Play → 인게임 전환 구현.
- 전투 종료 화면에 로비 복귀와 재도전 분기 연결.
- 메인 게임 플로우 확정: 실행 → 타이틀 → 로비 → 챕터 입장 → 웨이브 → 결과 → 로비.
- 결과 화면에서 로비 복귀와 같은 챕터 재도전 분기 정의.
- 프로젝트 문서 체계 시작: 기획서, 로드맵, 개발 기록, 의사결정 기록 생성.
- 현재 프로토타입 상태를 기준선으로 문서화.
- 앞으로 기능 변경 시 코드와 관련 문서를 함께 갱신하기로 함.
