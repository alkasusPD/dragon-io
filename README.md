# Sky Rogue

판타지 드래곤을 조종하는 세로형 탑뷰 로그라이트 슈팅 게임입니다.

## 실행

`index.html`을 Chrome, Edge 등 최신 브라우저로 엽니다.

## 조작

- 이동: WASD, 방향키, 화면 드래그
- 공격: 자동 사격
- 하강 회피: 화면 오른쪽 `하강` 버튼
- 일시정지: 화면 오른쪽 위 버튼

## 현재 구현

- 웨이브 진행 및 레벨업 강화 선택
- 단발탄, 직선 레이저, 원형 광역탄을 사용하는 적
- 3초 하강 회피와 2초 재사용 대기시간
- 윙맨 LV.1 좌측, LV.2 우측 소환
- 플레이어·적 12프레임 날갯짓 애니메이션
- HTML5 Canvas 기반 PC·모바일 브라우저 구동

## 캐릭터 자산 규격

- 스프라이트 시트: 가로 12프레임
- 프레임 크기: 256×256px
- 모든 판매형 플레이어 캐릭터가 동일 규격 사용
- 표시 크기와 충돌 판정은 이미지에서 분리
